## 使用方法

按照模板改一下account.json

在机器人的配置文件修改/加上IP或者域名配置，如果你的机器人配置文件没问题，应该会有下面三个值：

```python
PORT = 8080  # 本条请保持默认
IP = ''  # 修改为你的服务器ip,推荐修改
public_address = ''  # 修改为你的公网域名,推荐修改
```

将IP地址和域名填好，没有申请域名的不用填

## 功能：

1）自动过验证码

2）每个人可以绑定多个uid

3）每个uid可以设置不同的推送内容

4）内存记录击剑记录，重启bot会清空

#### 6）大部分指令仅支持群聊，支持私聊/群聊推送。
